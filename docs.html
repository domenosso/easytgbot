Документация TGEASYbot

TGEASYbot — это простой язык для создания Telegram-ботов. Ниже приведены все команды и примеры.

Команды

  • установка токена: Указывает токен бота.
    Пример: токен 123456:ABCDEF
  • обработка команд: Реагирует на команды, начинающиеся с "/".
    Пример: 
    команда /start
написать "Привет, я бот!"
конец
  • написать: Отправляет сообщение пользователю.
    Пример: написать "Привет!"
  • константы пользователя: Доступ к данным пользователя.
    Пример: id, user, name
  • рандом: Генерирует случайное число.
    Пример: рандом(1,10)
  • обычный текст: Обрабатывает текстовые сообщения.
    Пример: 
    текст от пользователя привет
написать "Привет! Чем могу помочь?"
конец

Примеры ботов

1. Бот, выдающий информацию о пользователе

установка токена
токен ВАШ_ТОКЕН

команда /start
написать "Привет, {name}! Твой ID: {id}, имя пользователя: {user}"
конец

Этот бот приветствует пользователя и показывает его данные.

2. Рандомайзер

установка токена
токен ВАШ_ТОКЕН

команда /random
написать "Случайное число: {рандом(1,100)}"
конец

Этот бот генерирует случайное число от 1 до 100 по команде /random.

3. Эхо-бот

установка токена
токен ВАШ_ТОКЕН

текст от пользователя любой_текст
написать "Ты написал: {любой_текст}"
конец

Этот бот повторяет любое сообщение пользователя.
